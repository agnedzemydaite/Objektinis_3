Detailed Description

Klasė 'studentas', paveldima iš klasės 'zmogus'.

Klasė leidžia saugoti studento informaciją:

  • vardą ir pavardę (paveldima iš zmogus),
  • pažymius,
  • egzamino rezultatą,
  • galutinį balą (vidurkį ir medianą),
  • namų darbų skaičių,
  • pažymių sumą.
Note
Klasėje įgyvendinta Rule of Three (destruktorius, copy konstruktorius, copy assignment).
